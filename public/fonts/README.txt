Fuente "Most Wanted"
====================

Para usar la fuente Most Wanted en la sección de equipos (The International / SEP 11-14, 2025):

1. Descarga la fuente Most Wanted (por ejemplo desde freefontdl.com u otra fuente legal).
2. Convierte el archivo a WOFF2 si es necesario (p. ej. con https://cloudconvert.com/ttf-to-woff2).
3. Guarda el archivo como: MostWanted.woff2
4. Colócalo en esta carpeta: public/fonts/

Mientras no haya archivo aquí, se usará Bebas Neue (estilo similar) como fallback.
